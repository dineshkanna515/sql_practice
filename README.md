Daily SQL practice as part of my Data Engineering preparation.

## Day 1 — SQL Basics
- SELECT, WHERE, ORDER BY, DISTINCT
- DISTINCT dedups on the full set of selected columns, not a single column
- WHERE vs HAVING: WHERE filters rows before grouping; HAVING filters groups after
- Clause execution order: FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY
